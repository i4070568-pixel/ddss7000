# 대덕산업(주) 홈페이지

GitHub Pages 배포용 정적 홈페이지입니다.

## 업로드 방법
1. GitHub에서 새 저장소 생성
2. 이 폴더의 모든 파일(index.html, assets/, .nojekyll)을 저장소 루트에 업로드
3. Settings > Pages > Build and deployment > Source를 `Deploy from a branch`로 선택
4. Branch를 `main`, Folder를 `/ (root)`로 선택 후 Save
5. 배포 주소: `https://아이디.github.io/저장소이름/`

## 파일 구조
- `index.html`
- `assets/`
- `.nojekyll`

## 참고
- 모든 이미지 경로는 GitHub Pages용 상대경로로 정리되어 있습니다.
- 문의 폼, 실제 다운로드 파일 연결 등은 추가 개발로 확장 가능합니다.
